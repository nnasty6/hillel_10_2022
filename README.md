## Quick start

### Install debs

```bash
# Install pipenv
pip install pipenv

# Activate virtual env
pipenv shell

# Install deps
pipenv sync
```
#### Additional
```bash
# Regenerate Pipfile.lock file
pipenv lock

# Pipenv lock & pipenv sync
pipenv update
```
